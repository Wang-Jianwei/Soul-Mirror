# 心镜 · 设计原则 (Design Principles)

> 配合 [02-features.md](02-features.md) 一起读  
> 审美参考：谷崎润一郎 · 阴翳礼赞、铃木大拙 · 禅与生活、王阳明书信  
> 立论：[00-fountainhead.md](00-fountainhead.md)

---

## 总原则

> **心镜是镜子，不是橱窗。**
> **照见，不是展示。**

## 一、审美基调：暗澹

谷崎润一郎在《阴翳礼赞》中说：

> "美不存在于物体，而存在于物体与物体之间的阴翳与明暗之中。"

心镜的视觉：
- **暗**为主调，**淡**为辅
- 大量留白，不压迫
- 字体偏大，行距偏宽
- 颜色不超过 3 种

## 二、配色

```css
/* 背景 */
--bg-primary:   #1A1A1A;   /* 深墨 */
--bg-card:      #2A2A2A;   /* 次墨 */
--bg-elevated:  #3A3A3A;   /* 浮起 */

/* 文字 */
--text-primary: #E8E2D5;   /* 米白 */
--text-muted:   #888880;   /* 淡灰 */
--text-faint:   #555550;   /* 几乎隐去 */

/* 强调（仅用于关键节点） */
--accent:       #C2A878;   /* 淡金 */
```

**永远不用的颜色**：
- 蓝紫渐变（AI slop）
- 鲜艳红 / 橙（焦虑贩卖）
- 高饱和"激励"色

## 三、字体

| 用途 | 字体 | 理由 |
|------|------|------|
| 标题 | Noto Serif SC | 衬线，有温度，像书法 |
| 正文 | Noto Sans SC | 无衬线，清晰 |
| 数据 | SF Mono / 等宽 | 静，不抢戏 |

字号：
- 主标题 28-32pt
- 正文 16-17pt（比行业标准大 1-2pt）
- 辅文 13-14pt

## 四、命名约定

**用中文意象，不用英文术语**

| 不要 | 要 |
|------|-----|
| "Streak" | "连续" |
| "Onboarding" | "入门" |
| "Daily Question" | "良知一叩" |
| "Mood Tracker" | "此刻心安" |
| "Decision Card" | "决策卡" |
| "Nature Map" | "本性图谱" |
| "Notification" | 不用（零推送） |
| "Achievement" | 不用 |
| "Leaderboard" | 不用 |

**不堆功能名词**：一个按钮只表达一件事。

## 五、交互原则：慢 / 静 / 简 / 退

### 1. 慢
- 页面切换 300ms 内完成
- 但用户**主动**快，心镜**被动**慢
- 不抢用户的注意力

### 2. 静
- 没有红点
- 没有未读数字
- 没有"3 条新消息"
- 看到的都是用户已经看过的

### 3. 简
- 一个屏幕只表达一个意思
- 按钮不超过 3 个
- 没有"立即升级 Pro"

### 4. 退
- 永远比用户少走一步
- 永远比用户少说一句
- 心镜的最高赞美："我不再需要它了"

## 六、伦理原则（产品红线）

| 原则 | 体现 |
|------|------|
| **零推送** | 用户主动打开 |
| **本地优先** | 敏感数据不上云（V1 完全本地） |
| **可彻底删除** | 一键清空 + 二次确认 + 不可恢复 |
| **不卖数据** | 商业模式不依赖数据 |
| **不焦虑贩卖** | 不发"你已经 3 天没记录了" |
| **不制造比较** | 无公开排行榜、无点赞 |
| **不输出"成功"** | 不说"恭喜你又坚持了 1 天" |

**违背任一条 = 产品失败，与技术无关。**

## 七、文案语调

参考：王阳明书信、禅宗公案、谷崎润一郎随笔

**避免**：
- "今天也要加油鸭！"
- "恭喜你又坚持了 1 天！"
- "你错过了今天的题目"
- "你不是一个人在战斗"（假共情）

**倾向**：
- "此刻，你的心安吗？"
- "本心在说：..."
- "这是本月第 3 次『知识焦虑』"
- "这周，你放下了什么？"
- "原来你是这样。"

**不主动用 emoji**。语气克制、温和、坚定。

## 八、动效原则

- 必要的微动效：30-200ms
- **避免** 弹跳 / 闪烁 / 振动（焦虑诱因）
- **避免** 复杂的进场动画
- **避免** "解锁成就"特效

## 九、信息密度

- 一屏不超过 3 个可操作元素
- 文字段落不超过 5 行
- 不显示"还有 N 条待办"
- 关键信息（"此刻心安"、"良知一叩"）默认在首屏可见

## 十、参考的人

- **谷崎润一郎** —— 阴翳礼赞，物的暗澹，审美主轴
- **铃木大拙** —— 禅与生活，文案克制，氛围主轴
- **王阳明** —— 书信体、不浮华、言之有物
- **Naoto Fukasawa** —— 极简产品设计，"Without Thought"
- **Dieter Rams** —— "Less but better"
- **小林一茶** —— 俳句的减法

**不参考**：Web 2.0 时代的 SaaS 模板，3D 拟物，赛博朋克，禅修 app 的"白噪音 + 引导语"范式。
