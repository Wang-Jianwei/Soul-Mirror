# 心镜 · 用户旅程 (User Journey)

> 配合 [02-features.md](02-features.md) 一起读  
> 哲学立基：阳明 · 事上练

---

## 总览

```
发现 → 首次打开 → 第一天 → 第一周 → 第一个月 → 第一年
 ↓        ↓         ↓        ↓         ↓          ↓
信任     入门      觉察     诊断     逍遥      本心档案
```

| 阶段 | 触点 | 心镜的角色 |
|------|------|----------|
| 发现 | 内容 / 朋友 | 沉默的镜子 |
| 首次打开 | App 启动 | 极简的入门 |
| 第一天 | 第一次记录 | 安静的 Aha |
| 第一周 | 连续打开 | 看见模式 |
| 第一个月 | 周观完成 | 开始有"流" |
| 第一年 | 年度档案 | 仪式感的回响 |

## 阶段 1 · 发现

**触点**：公众号 / 播客 / 朋友推荐

**用户心理**：
- "又一个自我提升 App？"
- "试试，但不抱希望"

**心镜的姿态**：
- 每一条内容都不带产品链接
- 只建立"这个团队在认真思考"的信任感
- 数字内容源风格：克制、有温度、有思辨

**目标**：让对的人"找上门来"，而不是"被拉进来"

## 阶段 2 · 首次打开

**流程**（3 步内完成）：

1. 欢迎页：「此刻，你的心安吗？」
2. 选择状态（不安 / 平静 / 喜悦 / 迷茫）
3. 直接进入"今日"页

**原则**：
- **无注册要求**（先体验，需要时再注册）
- **无功能介绍**（打开就是核心）
- **3 秒内**进入核心体验
- **不强制评估本性图谱**（V1 也是可选的）

## 阶段 3 · 第一天

**Aha moment**：

> 用户记录第一次「念头」  
> 输入："想买那门 2999 的课"  
> 心镜安静地说：「这是本月第 3 次『知识焦虑』」  
> 用户反应：「原来我是这样...」

**目标**：让用户第一次看见"被遮蔽的本心"——不是被告知，而是自己看见。

**这一刻的克制**：
- 不弹"恭喜你完成了第一次记录！"
- 不发勋章
- 不引导"分享给朋友"
- 心镜**退一步**，让用户独自消化

## 阶段 4 · 第一周

**关键行为**：
- 连续 5 天打开
- 完成第一次周观（四谛）

**仪式**：
- 周日晚自动生成"此心光明"小结（**不推送**）
- 用户主动打开，看见自己这一周的本心轨迹
- 不显示"连续 N 周"等激励性指标

**心镜的克制**：
- 周观不要做得太"漂亮"
- 不展示周环比、不显示趋势箭头
- 让数字自己说话

## 阶段 5 · 第一个月

**关键行为**：
- 完成本性图谱评估
- 至少一次无为清单
- 看见自己的本性分布

**留存机制（无为的留存）**：

| 行业惯例 | 心镜的做法 | 哲学依据 |
|---------|----------|---------|
| 推送通知 | 零推送 | 阳明 · 心只在主动时显现 |
| 红点 / 未读 | 无 | 老庄 · 无为 |
| 连续打卡奖励 | 无 | 孟子 · 义利之辨（不为奖励而活） |
| 周年活动 | 每年 12/31 自动生成档案 | 阳明 · 此心光明 |

持续使用的唯一理由：**内在的安宁**。

## 阶段 6 · 第一年

**关键事件**：
- 12/31 自动生成"此心光明"年度档案
- 邀请用户导出 PDF / 文本
- **不强制分享**

**目标**：
- 让用户感到"我变了"——不是心镜让他变了，是他自己
- 让用户成为"老用户"——不是因为积分，而是因为认同

## 流失用户

**心镜允许用户离开。**

**心镜的态度**：

- 用户用得少 = 用户的本心不需要心镜
- 用户离开 = 用户已经在用自己的方式活
- 心镜的最高追求：用户不再需要心镜

**这与所有同行产品相反**——同行追求留存，心镜追求让用户能离开。

## 新用户引导的 4 个原则

1. **不教** —— 没有人教过你如何使用镜子
2. **不解释** —— 第一次打开就进入核心
3. **不诱导** —— 没有任何"立即体验 Pro"按钮
4. **不留客** —— 用户离开，我们祝福

## 关键 Aha moment 排序

按重要性排：

1. **第一次"被看见"**（念头被识别为重复模式）
2. **第一次周观**（看见一周的本心轨迹）
3. **第一次本性图谱**（看见自己的本性分布）
4. **第一次致良知对话**（V2 才会有，AI 时代的关键 aha）
5. **第一次年度档案**（长期承诺的回报）

## 反模式：心镜避免的引导

| 反模式 | 心镜的做法 |
|--------|----------|
| 新手教程（5 屏引导） | 打开即核心 |
| "完善个人资料" | 默认匿名 |
| "邀请好友得 Pro" | 无邀请奖励 |
| "你已坚持 7 天！" | 静默 |
| "升级 Pro 解锁更多" | 永远在主功能内，不卖 |
