## 一句话

**陪你看见本心，陪你走出屏幕，把心活出来。**

## 哲学立基

心镜的全部设计，从一份判断出发：

> 人类从古至今，有 7 项从未改变的本性。孟子、佛家、阳明心学、老庄——四套跨越两千年的智慧体系，全部锚定这份"不变"来谈人该如何活。

心镜不是 productivity 工具，不是 mood tracker，不是冥想 app。

它是一面**镜子**，更是一位**道友**：照见你已经拥有、却常常被遮蔽的良知本心；陪你走出屏幕，把本心活成你这个人。

立论详尽推导见 [00-思想源泉.md](00-思想源泉.md)。本文档只交代从立论到产品的桥梁。

## 使命

**在焦虑时代，让人把本心活出来。**

三个动词，三个层次：

| 看见 | 走出 | 活出来 |
|------|------|--------|
| 在 app 内照见 | 把本心带进生活 | 让本心成为你这个人 |

只看见不够——**没走出 app 的知，等于未知**。  
只走出去不够——**没在生活里持续地活，等于没真正修**。

> 「未有知而不行者。知而不行，只是未知。」 —— 王阳明

## 设计形状：道友，不是道场

这是这一版最重要的判断：

| 道场（错的） | 道友（对的） |
|---|---|
| app 100%，生活 0% | app 30%，生活 70% |
| 修行发生在屏幕里 | 修行发生在事件里 |
| 功能 = 流程 | 功能 = 触点 + 见证 + 提醒 + 退让 |
| 用户用 app 完成任务 | 用户用 app 后回到生活 |

> 王阳明说"事上练"，不是"屏上练"。  
> 老庄说"道法自然"，是顺势而动，不是顺势而躺。

### app 的四个角色

1. **轻触点**——每个关系线一个最小入口
2. **见证者**——记下用户回来说的话
3. **提醒者**——节气、物件、问题、一行
4. **退让者**——在重大事件后主动消失

## 一面镜子的四个属性

| 属性 | 解释 |
|------|------|
| **镜子，不是老师** | 照见你本来就有的，不灌输你本没有的 |
| **照见，不是指导** | 给你看，不替你选 |
| **行，不是想** | 陪你想清楚，催你去事上练 |
| **还原，不是优化** | 找回被遮蔽的良知，不是变成"更好的你" |

## 五大关系线

心镜围绕**五个关系 + 一个元关系**展开——不是五个功能模块，是五种**生活的连接**：

| 关系 | 心镜陪用户做的事 | 哲学立基 |
|------|----------------|----------|
| **人与事** | 在真实事件里练 | 阳明 · 事上练 |
| **人与物** | 用一个物件锚定修行 | 老庄 · 道法自然 |
| **人与人** | 与一个同行者互相照见 | 阳明 · 讲学 |
| **人与心** | 与一个不让你逃的问题共处 | 阳明 · 良知 |
| **人与时** | 跟随节气与日出的节奏 | 老庄 · 道法自然 |
| **人与无** | 留白、沉默、退出 | 老庄 · 无为 |

> 阳明心学的核心是"事上练"。心镜的所有设计，都是为了让用户**在事件中活出本心**。app 是陪练，不是替代。

## 关键差异化：心镜不做什么

| 行业惯例 | 心镜不做 |
|---------|---------|
| 推送通知 | 零推送 |
| 算法推荐 | 无信息流 |
| 社交排名 | 无点赞、无等级 |
| 游戏化激励 | 无勋章、无积分 |
| 卖用户数据 | 本地优先，可彻底删除 |
| 焦虑贩卖 | "你已经 3 天没记录了" |
| 任务清单 | 1 件具体的事，不是 TODO |
| 持续占位 | 在重大事件后主动消失 |
| 假陪伴 | AI 是镜子，不是伴侣 |
| 公众曝光 | 1 个同行者，不是社区 |
| 每日打卡 | 跟随节气和心，不跟打卡 |
| "立即升级 Pro" | 永远在主功能内，不卖 |

**违背任一条 = 产品失败，与技术无关。**

## 目标用户

**主要**：25-40 岁知识工作者 / 创意从业者 / 中层管理者

**心理特征**：
- 感到效率工具的"够用但没意义"
- 买过 20+ 在线课程，完成率不到 10%
- 知道自己"应该"做什么，但不想做
- 在"被优化"和"摆烂"之间找不到第三条路

**典型场景**：

> 晚上 10 点，躺在床上刷手机，突然想买一门 2999 的课。  
> 打开心镜，记下：「想买 AI 课，害怕被淘汰」。  
> 心镜安静地说：「这是本月第 3 次『知识焦虑』」。  
> 关闭 app。把"今日一行"想清楚："明天中午给妈打个电话。"  
> 睡觉。
> 
> 明天中午，真的打了。

**注意：app 在第 4 步就退出了。真正的修行是"明天中午真的打了"。**

## 终极愿景

> **此心光明，亦复何言。**

——王阳明临终语。心镜陪用户走过的，不是"用了多少 app"，而是"在多少事上练过"。
