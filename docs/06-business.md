## 核心原则

> **不赚焦虑的钱，不赚注意力的钱，不赚成瘾的钱。**

信任即壁垒。在一个贩卖焦虑的时代，**"不打扰"本身就是稀缺品。**

## 收入结构

| 层级 | 产品 | 定价 | 逻辑 |
|------|------|------|------|
| **免费层** | 觉察镜 + 简单统计 | ¥0 | 降低门槛 |
| **订阅层** | 心镜 Pro（完整五大镜） | ¥38/月 或 ¥298/年 | 深度用户付费 |
| **服务层** | 心镜·师 1v1 对话 | ¥500-2000/次 | 高净值用户 |
| **企业层** | 心镜·组织 团队版 | 按需定价 | B 端拓展 |

### 订阅层（Pro）详细

**包含**：
- 完整五大镜
- 本性图谱年度更新
- 致良知镜无限次（V2+）
- 数据导出

**定价心理**：

> 一杯精品咖啡 = 一个月的自我觉察

**不打折**：不为"限时 5 折"折损信任。

## 关键差异化：心镜不做什么

| 行业惯例 | 心镜不做 |
|---------|---------|
| 推送通知轰炸 | 零推送 |
| 算法推荐 | 无信息流 |
| 社交裂变拉新 | 无分享按钮 |
| 游戏化积分排名 | 无等级、无勋章 |
| 卖用户数据 | 本地优先，可彻底删除 |
| 焦虑留存话术 | "你已经 3 天没记录了" |
| 限时折扣 | 不为转化折损气质 |

**违背任一条 = 产品失败。**

## 核心竞争壁垒

| 壁垒 | 说明 | 可复制性 |
|------|------|---------|
| **哲学框架** | 孟子-佛家-阳明-老庄的融合体系 | 低，需深度理解 |
| **产品气质** | 反焦虑、反成瘾的设计伦理 | 中，可被模仿但难坚持 |
| **用户信任** | 数据可删、零推送、不贩卖焦虑 | 中，需长期坚持 |
| **内容资产** | 「不变的人性」故事库 | 中，持续积累 |
| **节奏** | 慢增长、慢迭代 | 低，多数团队熬不住 |

## 增长策略：慢增长

> **不是用户越多越好，是用户越"对"越好。**
> **前 1000 个用户，比 100 万更重要。**

### Phase 1 · 种子期（0-6 个月）

**目标**：找到 100 个"对的人"

**做法**：
- 内容先行：「不变的人性」系列长文 / 播客
- 邀请制内测：必须填申请问卷
- 每周 1v1 访谈（5 个 / 周）

**里程碑**：50 人愿意付费（哪怕产品很粗糙）

### Phase 2 · 口碑期（6-12 个月）

**目标**：100 → 1,000

**做法**：
- 「此心光明」用户故事征集（不美化）
- 跨界合作：独立书店 / 心理咨询师 / 慢生活播客
- 老用户邀请制（无现金奖励，只送《心镜使用手记》）

**逻辑**：愿意为"小册子"邀请朋友的人，才是"对的人"。

### Phase 3 · 扩散期（12-24 个月）

**目标**：1,000 → 10,000

**做法**：
- 「反焦虑」事件营销：发布《中国年轻人焦虑报告》
- 不找大 V，找"深 V"（哲学博主 / 心理咨询师）
- 企业版试点（互联网大厂 / 创业团队）

## 关键指标

| 阶段 | 北极星指标 | 目标 |
|------|-----------|------|
| 种子期 | "被触动"率 | > 60% |
| 口碑期 | 自然推荐率 | > 30% |
| 扩散期 | 付费转化率 | > 15% |

**不看的指标**：
- 下载量
- 日活 / 月活
- 使用时长
- 留存曲线（用"被触动"代替）

| 年份 | 付费用户 | ARR | 成本 | 利润 |
|------|---------|-----|------|------|
| 第 1 年 | 10,000 | ¥300 万 | ¥250 万 | ¥50 万 |
| 第 2 年 | 40,000 | ¥1,200 万 | ¥800 万 | ¥400 万 |
| 第 3 年 | 100,000 | ¥3,000 万 | ¥1,800 万 | ¥1,200 万 |

**关键判断**：第 1 年不追求利润，追求"找到对的用户"。

## 风险与应对

| 风险 | 应对 |
|------|------|
| 增长太慢，资金链断 | 控制团队规模，先验证再扩张 |
| 内容没人看 | 不追热点，坚持深度，相信长尾 |
| 用户留存低 | 强化"年度回顾"的仪式感 |
| 竞品抄袭 | 速度不是壁垒，气质和信任是 |
| AI 成本过高 | 本地模型分担，优化 Prompt 长度 |
| 团队跑不动"慢" | 文化筛选 > 能力筛选 |

## 终极愿景

> **不是成为最大的 App，是成为最"真"的 App。**

用户留下的不是数据，是**觉醒的轨迹**。
