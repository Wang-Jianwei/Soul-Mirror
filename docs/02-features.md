## 设计哲学

心镜的功能**不来自市场调研或用户故事**，而来自四套哲学传统的功能映射：

| 学派 | 给心镜的礼物 |
|------|-------------|
| **7 不变本性** | 评估的 7 个维度 |
| **孟子 · 性善论 / 四端** | 良知日课的内容来源 |
| **孟子 · 义利之辨** | 诊断镜的核心问题 |
| **佛家 · 四圣谛** | 周观的结构 |
| **佛家 · 戒定慧** | 修行的三阶 |
| **佛家 · 五毒** | 情绪的 5 种类型 |
| **阳明 · 良知** | 致良知镜的存在理由 |
| **阳明 · 知行合一** | 决策辅助的根本逻辑 |
| **阳明 · 事上练** | 全部功能的隐含前提 |
| **老庄 · 道法自然** | 自然节律、节气提示 |
| **老庄 · 无为** | 显式的"不做清单" |
| **老庄 · 心斋坐忘** | 空心冥想 |

每一条功能都可以在 fountainhead 里找到根源。

## 五大功能镜

### 1. 觉察镜 · 每日 5 分钟

**哲学立基**：阳明 · 良知、佛家 · 观

**包含**：

| 模块 | 内容 | 节奏 |
|------|------|------|
| **此刻心安** | 不安 / 平静 / 喜悦 / 迷茫 状态打卡 | 每日 |
| **良知一叩** | 今日，本心说了什么？ | 每日，365 题轮换 |
| **念头一记** | 升起又灭去的念头 | 随时 |
| **身体一感** | 身体此刻的感受（可选） | 配合念头 |

**关键设计**：
- 零推送，打开才有
- 3 秒内完成一次记录
- 念头不评分、不判断，只归档
- 365 问 题库是良知一叩的内容源

**对应学校**：阳明 · 心即理 / 佛家 · 观

---

### 2. 诊断镜 · 每周 15 分钟

**哲学立基**：佛家 · 四圣谛、孟子 · 义利之辨、佛家 · 五毒

**包含**：

| 模块 | 核心问题 |
|------|---------|
| **四谛周观** | 苦：最让你痛的是什么？集：根源在哪？灭：放下会怎样？道：下一步修行？ |
| **义利之辨** | 本周，自利与公义的天平偏了几次？ |
| **五毒观** | 贪 / 嗔 / 痴 / 慢 / 疑 各出现几次？ |

**关键设计**：
- 周日晚自动生成（**不推送**）
- 用户主动打开
- 简洁可视化，避免数据焦虑
- 不展示"已完成周观数"等激励性指标

**对应学校**：佛家 · 四圣谛 / 孟子 · 义利之辨

---

### 3. 致良知镜 · 随时可用

**哲学立基**：阳明 · 事上练 / 知行合一

**包含**：

| 模块 | 内容 |
|------|------|
| **事上练对话** | 写下一个纠结，AI 帮你剥离情绪看清本心（苏格拉底式） |
| **知行合一镜** | 你想做的 vs 你做的 |
| **决策卡** | 生成 + 3 个月回顾（"当时本心怎么说"） |

**关键设计**：
- 不替用户做决定，只帮用户看清
- 对话 5-10 轮，温和、坚定、不评判
- 区分"恐惧 vs 真实危险"、"欲望 vs 真正需要"
- 决策卡是核心 artifact，可永久回看
- V1 不引入 AI（Phase 2 加入）

**对应学校**：阳明 · 致良知 / 知行合一

---

### 4. 逍遥镜 · 每月 30 分钟

**哲学立基**：老庄 · 无为 / 道法自然、7 不变本性

**包含**：

| 模块 | 内容 |
|------|------|
| **本性图谱** | 基于 7 不变本性的雷达图（每月更新） |
| **无为清单** | 这个月，你主动不做什么？显式的"不做"功能 |
| **心斋** | 10 分钟空心冥想（不引导注意力，引导放下） |
| **自然节律** | 跟随日出日落、节气的小提示（不制造任务） |

**关键设计**：
- 无为清单是显式功能（与"任务清单"对立）
- 心斋不引导注意力（不像 Headspace），而引导放下
- 自然节律不制造任务，只给提示
- 本性图谱不评分高低，只看见分布

**对应学校**：老庄 · 无为 / 道法自然

---

### 5. 本心档案 · 每季 / 每年

**哲学立基**：阳明 · 此心光明、孟子 · 求其放心

**包含**：

| 模块 | 内容 |
|------|------|
| **此心光明** | 季度 / 年度回顾（自动生成） |
| **本性流变** | 7 维本性的长期变化 |
| **关键决策回顾** | 当时本心怎么说？现在回头看？ |
| **数据导出** | PDF / 文本（你的觉醒轨迹） |

**关键设计**：
- 自动生成，仪式感强
- 可分享（不绑定账号）
- 离线可读，永久归档
- 每年 12/31 是一年一度的"心镜时刻"

**对应学校**：阳明 · 致良知

## 时间节奏

| 节奏 | 模块 | 时长 | 触发 |
|------|------|------|------|
| 每日 | 觉察镜 | 5 分钟 | 用户主动 |
| 每周 | 诊断镜 | 15 分钟 | 周日晚（**无推送**） |
| 随时 | 致良知镜 | 5-30 分钟 | 决策时刻 |
| 每月 | 逍遥镜 | 30 分钟 | 月初（**无推送**） |
| 每季/每年 | 本心档案 | 30 分钟 | 自动生成 |

**核心原则**：无推送，全部用户主动。**本心只在主动时显现。**

## V1 明确不做

为了"少即是多"，V1 明确不做：

| 不做 | 理由 |
|------|------|
| 逍遥社区 | 社交容易变成比较 |
| AI 心理陪伴 | AI 是镜子，不是伴侣 |
| 公开排行榜 | 比较是焦躁的根 |
| 课程 / 内容流 | 心镜不输入，只帮你整理自己 |
| 跨平台账号系统 | 本地优先，V1 不依赖云端 |
| 推送通知 | 用户主动，本心才显现 |
| 红点 / 未读数字 | 不会被"未完成"绑架 |

## 数据模型（核心）

```typescript
// 念头
interface Thought {
  id: string;
  content: string;
  type?: 'knowledge_anxiety' | 'material' | 'relationship' | 'achievement' | 'escape' | 'other';
  intensity?: number;        // 1-10
  bodySensation?: string;
  createdAt: Date;
  resolvedAt?: Date;
  satisfaction?: number;     // 1-10
}

// 良知日答
interface ConscienceLog {
  id: string;
  questionId: string;
  answer: string;
  createdAt: Date;
}

// 周观（四谛 + 义利 + 五毒）
interface WeeklyReview {
  id: string;
  weekStart: Date;
  suffering: string;       // 苦
  cause: string;           // 集
  cessation: string;       // 灭
  path: string;            // 道
  selfVsRight: number;     // -10 ~ +10
  poisons: {               // 五毒次数
    greed: number;
    hatred: number;
    delusion: number;
    pride: number;
    doubt: number;
  };
}

// 决策卡
interface DecisionCard {
  id: string;
  title: string;
  context: string;
  conversation: Message[];
  conclusion: string;
  chosenOption?: string;
  reviewAt: Date;          // 3 个月后
  reviewNotes?: string;
  createdAt: Date;
}

// 本性图谱
interface NatureMap {
  survival: number;        // 生存优先
  reproduction: number;    // 繁衍本能
  belonging: number;       // 群居归属
  pleasure: number;        // 趋乐避苦
  selfInterest: number;    // 自利利他
  curiosity: number;       // 好奇探索
  emotion: number;         // 共通情绪
  updatedAt: Date;
}
```

## MVP 范围（Phase 1）

**目标**：验证 100 个种子用户愿意每日打开心镜

**包含**：
- 觉察镜（此刻心安 / 良知一叩 / 念头一记）
- 简单周统计（念头数、状态分布）
- 一次性本性图谱（7 维评估）

**不包含**（留给 Phase 2/3）：
- AI 致良知镜
- 决策卡
- 逍遥镜
- 本心档案
- 多设备同步
- 云端备份
