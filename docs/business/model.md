**不赚焦虑的钱，不赚注意力的钱，不赚成瘾的钱。**

## 订阅层（Pro）详细设计

**定价：¥38/月，¥298/年（省¥158）**

- 完整三层循环（觉察→回归→践行）

### 定价心理

> 一杯精品咖啡 = 一个月的自我觉察

---

## 关键差异化：「不做什么」

| 行业惯例 | 心镜不做 |
|---------|---------|
| 推送通知轰炸 | 零推送，用户主动打开 |
| 算法推荐内容 | 无信息流，无无限滚动 |
| 社交裂变拉新 | 无分享按钮，口碑自然传播 |
| 游戏化积分排名 | 无等级、无勋章、无竞争 |
| 卖用户数据 | 本地优先，数据可完全删除 |

**信任即壁垒**——在一个贩卖焦虑的时代，「不打扰」本身就是稀缺品。

---

## 增长策略：慢增长

**不是用户越多越好，是用户越「对」越好。**

### 渠道

- **内容**：公众号/播客讲「不变的人性」故事，不推销产品
- **合作**：与心理咨询师、人生教练、瑜伽/冥想工作室联名
- **口碑**：老用户邀请制，控制节奏

### 目标

| 时间 | 目标 |
|------|------|
| 第1年 | 10,000 付费用户 |
| 第3年 | 100,000 付费用户，¥3,000万 ARR |
| 第5年 | 成为「反焦虑」生活方式的代名词 |

## 风险与应对

| 风险 | 应对 |
|------|------|
| 增长太慢，资金链断 | 控制团队规模，先验证再扩张 |
| 用户留存低 | 强化「年度回顾」的仪式感 |
| 竞品抄袭 | 速度不是壁垒，气质和信任是 |
| AI成本过高 | 本地模型分担，优化Prompt长度 |
