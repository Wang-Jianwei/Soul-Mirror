# 产品需求文档 (PRD)

## 产品概述

**Soul Mirror (心镜)** 是一个基于「不变人性」的个人觉醒产品。

---

## 用户画像

### 主要用户

- **年龄**: 25-40 岁
- **职业**: 知识工作者、创意从业者、管理者
- **特征**: 
  - 经常感到焦虑或迷茫
  - 对自我成长有需求
  - 不喜欢传统的效率工具
  - 追求内心的平静而非外在的成功

### 用户痛点

1. 被各种效率工具绑架，反而更焦虑
2. 不知道自己真正想要什么
3. 做决策时被恐惧和欲望裹挟
4. 缺乏对自我的深度理解

---

## 核心功能

### 1. 渴爱追踪 (Desire Tracker)

**用户故事**: 
> 作为一个经常冲动消费的用户，我想要记录自己的欲望，以便看清自己的模式。

**功能描述**:
- 快速记录当下的「渴爱」
- 标记渴爱类型（知识、物质、关系等）
- 追踪满足后的实际感受
- AI 识别重复模式

**验收标准**:
- [ ] 3 秒内完成记录
- [ ] 支持文字、语音输入
- [ ] 自动生成模式报告

### 2. 本性图谱 (Nature Map)

**用户故事**:
> 作为一个迷茫的职场人，我想要了解自己的本性倾向，以便找到适合自己的生活方式。

**功能描述**:
- 7 项本性评估问卷
- 雷达图可视化
- 本性相似者故事推荐
- 不适合路径预警

**验收标准**:
- [ ] 问卷完成时间 < 10 分钟
- [ ] 结果准确度高（用户认可 > 80%）
- [ ] 故事推荐相关度高

### 3. 良知决策 (Conscience Decision)

**用户故事**:
> 作为一个面临两难选择的人，我想要剥离情绪看清本心，以便做出不后悔的决定。

**功能描述**:
- AI 苏格拉底式提问
- 决策卡片生成
- 3 个月后回顾提醒
- 决策历史追踪

**验收标准**:
- [ ] 对话自然流畅
- [ ] 帮助用户看清核心矛盾
- [ ] 决策后满意度 > 70%

### 4. 逍遥社区 (Freedom Community)

**用户故事**:
> 作为一个感到孤独的用户，我想要看到跟我一样的人怎么活，以便获得勇气和启发。

**功能描述**:
- 基于本性相似度的匹配
- 无点赞、无排名的故事分享
- 共鸣功能（仅作者可见）
- AI 内容审核

**验收标准**:
- [ ] 社区氛围正向
- [ ] 无焦虑贩卖内容
- [ ] 用户参与度高

### 5. 年度回顾 (Year Review)

**用户故事**:
> 作为一个重视成长的用户，我想要回顾一年的心路历程，以便看到自己的变化。

**功能描述**:
- 自动生成年度数据报告
- 关键决策回顾
- 本性变化追踪
- PDF 导出

**验收标准**:
- [ ] 报告生成时间 < 5 秒
- [ ] 设计美观、有仪式感
- [ ] 用户分享率高

---

## 非功能需求

### 性能

- App 启动时间 < 2 秒
- 页面切换流畅，无卡顿
- 离线可用核心功能

### 安全

- 本地数据加密存储
- 云端传输 HTTPS
- 用户可彻底删除数据

### 可用性

- 零学习成本
- 深色模式默认
- 支持无障碍访问
