## 二、觉察模块「三层循环」

### 2.1 第一层：念头追踪

**入口**：点击「记录念头」

```
┌─────────────────────────────────────────┐
│  ← 念头追踪                             │
│                                         │
│  此刻，你在渴什么？                      │
│  ┌─────────────────────────────────┐   │
│  │ 想买那门 2999 的课...            │   │
│  └─────────────────────────────────┘   │
│                                         │
│  这是第几次出现这个渴？                  │
│  ○ 第一次  ○ 第 N 次                   │
│                                         │
│  上一次得到满足后，感觉持续了多久？       │
│  ○ 几小时  ○ 几天  ○ 记不清了          │
│                                         │
│  [ 查看历史模式 ]                       │
│                                         │
│  ─────────────────────────────────────  │
│                                         │
│  系统提示（非强制）：                    │
│  「你过去 3 次类似记录，                │
│    实际使用率：12%」                     │
│                                         │
│           [ 确认记录 ]                  │
│                                         │
└─────────────────────────────────────────┘
```

**记录后**：
- 不评分、不判断
- 仅归档，供后续模式识别

### 2.3 第三层：良知决策

**入口**：「我有纠结」

```
┌─────────────────────────────────────────┐
│  ← 良知决策                             │
│                                         │
│  写下你的两难：                          │
│  ┌─────────────────────────────────┐   │
│  │ 要不要接受这个 offer？           │   │
│  │ 钱多 30%，但要 996，搬家去上海   │   │
│  └─────────────────────────────────┘   │
│                                         │
│  [ 开始对话 ]                           │
│                                         │
│  ─────────────────────────────────────  │
│                                         │
│  系统：我们先剥离一下。                  │
│        这个决定里，有多少是恐惧？        │
│                                         │
│  你：害怕现在的工作不稳定               │
│                                         │
│  系统：如果完全不用担心钱和稳定，        │
│        你的本心会怎么选？                │
│                                         │
│  你：可能会选现在这个，有时间陪家人      │
│                                         │
│  系统：那么，这个「害怕不稳定」，        │
│        是真实的危险，还是被放大的恐惧？  │
│                                         │
│  [ 继续对话 ]  [ 结束，记录结论 ]        │
│                                         │
└─────────────────────────────────────────┘
```

**对话结束后**：
- 生成「决策卡片」：当时本心怎么说
- 归档，3 个月后可回顾

## 六、交互细节

| 场景 | 设计 |
|------|------|
| 启动 App | 无开屏广告，直接进入「今日」 |
| 加载等待 | 显示随机金句，非转圈圈 |
| 错误提示 | 「心镜暂时迷路了，请稍后再试」 |
| 空状态 | 「这里什么都没有，像你的心一样干净」 |
| 深色模式 | 默认深色，护眼，深夜使用无负担 |
