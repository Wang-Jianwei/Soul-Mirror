## 一句话

**帮你看见从未变过的本心，活成自己的人。**

## 哲学立基

心镜的全部设计，从一份判断出发：

> 人类从古至今，有 7 项从未改变的本性。孟子、佛家、阳明心学、老庄——四套跨越两千年的智慧体系，全部锚定这份"不变"来谈人该如何活。

心镜不是 productivity 工具，不是 mood tracker，不是冥想 app。  
它是一面**镜子**：照见你已经拥有、却常常被遮蔽的良知本心。

立论详尽推导见 [00-fountainhead.md](00-fountainhead.md)。本文档只交代从立论到产品的桥梁。

## 使命

**在焦虑时代，让人回到从未改变的本心。**

- 不帮你做更多
- 不帮你变更好
- 不优化你的人生

只帮你看清：**什么值得做，什么不值得做。**

## 一面镜子的三个属性

| 属性 | 解释 |
|------|------|
| **镜子，不是老师** | 照见你本来就有的，不灌输你本没有的 |
| **照见，不是指导** | 给你看，不替你选 |
| **还原，不是优化** | 找回被遮蔽的良知，不是变成"更好的你" |

## 关键差异化：心镜不做什么

| 行业惯例 | 心镜不做 | 理由 |
|---------|---------|------|
| 推送通知 | 零推送 | 用户主动打开，本心才会显现 |
| 算法推荐 | 无信息流 | 心镜不是消费内容的地方 |
| 社交排名 | 无点赞、无等级 | 比较是焦躁的根 |
| 游戏化激励 | 无勋章、无积分 | 内在驱动，不是外部奖励 |
| 卖用户数据 | 本地优先，可彻底删除 | 信任即壁垒 |
| 焦虑贩卖 | "你已经 3 天没记录了" | 戒律：不为留存制造焦虑 |

## 目标用户

**主要**：25-40 岁知识工作者 / 创意从业者 / 中层管理者

**心理特征**：
- 感到效率工具的"够用但没意义"
- 买过 20+ 在线课程，完成率不到 10%
- 知道自己"应该"做什么，但不想做
- 在"被优化"和"摆烂"之间找不到第三条路

**典型场景**：

> 晚上 10 点，躺在床上刷手机，突然想买一门 2999 的课。  
> 打开心镜，记下：「想买 AI 课，害怕被淘汰」。  
> 心镜安静地说：「这是本月第 3 次『知识焦虑』」。  
> 愣一下，放下手机，睡觉。

## 关键判断：心镜是工具还是修行？

**心镜表面是工具，骨子里是修行。**

差别在于：

| 工具 | 修行 |
|------|------|
| 追求效率 | 承认缓慢 |
| 给答案 | 指方向 |
| 越多越好用 | 越少越够用 |

我们的产品原则：**做"够用即止"的工具，让人回到不需要工具的状态。**

心镜的最高赞美——"我不再需要它了"。
