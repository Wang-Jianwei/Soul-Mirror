# Agent Context - 项目上下文

## 🎯 项目概述

**Soul Mirror (心镜)** 是一个基于「不变人性」的个人觉醒产品。

### 核心使命

帮助用户：
1. **觉察** - 看清自己的渴爱与恐惧
2. **回归** - 识别本性，找到适合的路径
3. **践行** - 事上练，致良知

### 不变人性（7项）

1. 生存优先 - 趋利避害的本能
2. 繁衍本能 - 延续基因、亲缘利他
3. 群居归属 - 渴望联结、害怕孤独
4. 趋乐避苦 - 追求愉悦、规避痛苦
5. 自利利他 - 双重本性的平衡
6. 好奇探索 - 对未知的掌控欲
7. 共通情绪 - 喜怒哀乐的底层机制

## 📚 哲学基础

### 四家融合

| 学派 | 核心思想 | 在心镜中的体现 |
|------|---------|---------------|
| **孟子** | 性善论、四端说 | 恻隐之心、良知决策 |
| **佛家** | 苦谛、无我 | 念头记录、欲望解构 |
| **阳明心学** | 心即理、知行合一 | 事上练、致良知 |
| **老庄** | 道法自然、无为 | 本性图谱、逍遥社区 |

## 🏗️ 技术架构

```
客户端 (React Native)
    ↓
本地 SQLite (敏感数据)
    ↓
云端 (Supabase + AI API)
```

### 核心模块

- **念头记录** - 欲望记录与模式识别
- **本性图谱** - 7项本性评估与可视化
- **良知决策** - AI 引导的决策辅助
- **逍遥社区** - 本性相似者的故事分享
- **年度回顾** - 「此心光明」PDF 生成

## ⚠️ 约束条件

### 产品约束

- 零推送通知
- 无算法推荐
- 无社交裂变
- 无游戏化排名
- 数据可完全删除

### 技术约束

- 本地优先存储
- 支持离线使用
- 端到端加密
- 开源核心代码
