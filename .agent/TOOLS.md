# TOOLS.md - 本地笔记

Skills 定义工具的_工作方式_。这个文件是你的_特定_—— 那些你独特设置才有的东西。

## 写什么

比如：

- 摄像头名字和位置
- SSH 主机和别名
- TTS 偏好语音
- 扬声器/房间名字
- 设备昵称
- 任何环境相关的东西

## 例子

```markdown
### 摄像头

- 客厅 → 主区域，180° 广角
- 入户门 → 入口，运动触发

### SSH

- 家用服务器 → 192.168.1.100，用户：admin

### TTS

- 偏好语音："Nova"（温暖、略英式）
- 默认扬声器：厨房 HomePod
```

## 为什么要分开？

Skills 是共享的。你的设置是你的。把它们分开，你更新 skill 时不会丢失笔记，分享 skill 时也不会泄露基础设施。
