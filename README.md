# Soul Mirror (心镜) v0.1

> 一个帮你记录念头、看见模式的极简工具。

## 功能

- **今日首页**: 心安状态选择、每日一问、本周统计
- **觉察**: 念头记录（内容+强度+类型）
- **念头历史**: 按日期分组的记录列表
- **每日一问**: 365 日问题库，记录回答
- **我的**: 数据统计、数据导出(JSON)、清空数据

## 技术栈

- React Native + TypeScript
- Expo
- SQLite (本地存储)
- Zustand (状态管理)
- React Navigation
- React Native Paper

## 安装

```bash
npm install
npm start
```
