# 心镜 (Soul Mirror)

> 一个基于不变人性的个人操作系统
> 
> 不是工具，是一面让你看清自己的镜子。

---

## 核心理念

人类有七个从未改变的本性：

1. **生存优先** - 趋利避害的本能
2. **繁衍本能** - 延续基因、亲缘利他
3. **群居归属** - 渴望联结、害怕孤独
4. **趋乐避苦** - 追求愉悦、规避痛苦
5. **自利利他** - 双重本性的平衡
6. **好奇探索** - 对未知的掌控欲
7. **共通情绪** - 喜怒哀乐的底层机制

心镜融合孟子、佛家、阳明心学、老庄四家思想，基于这些不变本性，帮助用户：

- **觉察** - 看清自己的渴爱与恐惧
- **回归** - 识别本性，找到适合的路径
- **践行** - 事上练，致良知

## 产品架构

### 三层循环

```
┌─────────────────────────────────────┐
│  第一层：觉察（对应佛家"苦谛"）        │
│  记录 → 看清 → 解构                   │
│  "我在渴什么？这渴能解渴吗？"          │
├─────────────────────────────────────┤
│  第二层：回归（对应老庄"顺性"）        │
│  识别本性 → 匹配路径 → 无为而治        │
│  "这是我真正想要的，还是被灌输的？"     │
├─────────────────────────────────────┤
│  第三层：践行（对应阳明"致良知"）      │
│  事上练 → 知行合一 → 此心光明         │
│  "此刻，我的本心怎么说？"              │
└─────────────────────────────────────┘
```

### 核心模块

| 模块 | 功能 | 哲学基础 |
|------|------|---------|
| **渴爱追踪** | 记录欲望→得到→失落的完整周期 | 佛家"趋乐避苦" |
| **本性图谱** | 识别7项本性强度，推荐适合活法 | 老庄"自然之性" |
| **良知决策** | 两难时刻，剥离私欲，问出本心 | 阳明"事上练" |
| **逍遥社区** | 无点赞、无排名，本性相似者的故事 | 庄子"齐物" |
| **共情练习** | 每日一题，训练推己及人 | 孟子"恻隐之心" |

## 产品原型

### 首页「今日」

打开 App，先问：

> **此刻，你的心安吗？**

四个选项：不安 😔 / 平静 😌 / 喜悦 😊 / 迷茫 😶

然后进入「今日一问」，来自365题题库。

### 底部导航（仅4项）

- **今日** - 心安检查 + 每日一问
- **觉察** - 渴爱追踪、良知决策
- **本性** - 本性图谱、逍遥社区
- **我的** - 年度回顾、设置

**无通知、无红点、无数字提醒**

## 冷启动策略

### 原则

> 前1000个用户，一个一个聊出来。

### 三个阶段

1. **种子期（0-6月）**
   - 内容先行：公众号、播客、小红书
   - 邀请制内测：必须填写申请问卷
   - 每周1v1访谈

2. **口碑期（6-12月）**
   - 「此心光明」用户故事计划
   - 跨界合作：书店、瑜伽工作室、心理咨询师
   - 老用户邀请：无现金奖励，赠电子手册

3. **扩散期（12-24月）**
   - 「反焦虑」宣言事件营销
   - KOL策略：找「深V」而非大V
   - 企业版试点

## 竞品分析

| 产品 | 模式 | 心镜差异 |
|------|------|---------|
| Headspace | 引导冥想 | 不止放松，指向觉醒 |

**定位**：Headspace帮你放松，KnowYourself帮你理解，心镜帮你觉醒。
